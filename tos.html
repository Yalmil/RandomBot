Términos de Servicio - RandomBot

Resumen rápido

Usando RandomBot aceptas estos términos. Respeta las reglas de Discord y no abuses de las funciones del bot

1. Aceptación de los términos

Estos términos regulan el uso de RandomBot. Si no estás de acuerdo, no uses el bot

2. Uso permitido

Puedes usar el bot para las funciones descritas en su documentación. Está prohibido usarlo para actividades ilegales, spam o violaciones de privacidad

3. Contenido y responsabilidad

El propietario del bot no se hace responsable por el contenido generado por usuarios. Eres responsable de tus acciones y de las sanciones en servidores o por Discord

4. Privacidad y datos

El bot puede recopilar datos mínimos necesarios para funcionar (por ejemplo, identificadores de usuario y servidor). No compartiremos datos con terceros salvo por requisitos legales o integraciones explícitas

5. Prohibiciones

No uses el bot para: explotaciones, doxxing, suplantación, ataques o distribuir malware. Tampoco para evadir bans o abusar de APIs

6. Modificaciones y terminación

Podemos actualizar estos términos en cualquier momento. El uso continuado del bot implica aceptación de los cambios. Nos reservamos el derecho a suspender el servicio o el acceso a usuarios que incumplan

7. Limitación de responsabilidad

En la máxima medida permitida por la ley, el propietario no será responsable por daños indirectos, pérdida de datos o lucro cesante derivados del uso del bot

8. Contacto

Para dudas, avisos de abuso o solicitudes de datos, contacta al propietario vía el servidor oficial o el correo asociado al proyecto
